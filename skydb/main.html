SkyDB - A ComputerCraft: Tweaked Database format

SkyDB is a filetype with is read and loaded with the AdvFS library

It is loaded with `AdvFS.tblLoad`, and saved with `AdvFS.tblSave`.

    
      AdvFS = require("AdvFS")
      data = AdvFS.tblLoad("database.skydb")
      data[#data + 1] = "This is the new last entry in this database!"
      AdvFS.tblSave(data,"database.skydb")
