The File Library

The file library contains many useful functions for working with files

Functions:

file.split(sText:string, sSeperator:string): table
-> Splits 'sText' by 'sSeperator' and returns a table of the split text.
file.countLines(sPath:string): number
-> Counts the number of lines in 'sPath' and returns a number of the lines.
file.loadGrpLines(sPath:string)
-> Loads the specified graphics file and draws it to the screen.
file.loadAppGraphics(sGraphicPath:string, sSettingsPath:string, sAppName:string)
-> Loads the specified graphics file, and uses the settings path and app name to determine offset.
file.getSize(sPath:string): number
-> Recursively goes through a directory to get the size of every file within the directory, and returns the size of it all.
