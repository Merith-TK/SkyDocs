The GPSwrapper Library

The gpswrapper library is a wrapper around the gps library, but works in an emulator. If the gps library is not present, this returns 1,1,1

Functions:

gpswrapper.gpslocate(nTimeout)
-> This checks to see if the gps library exists, if so, it pings gps servers, if not, it returns 1,1,1

 gpswrapper.gpsfloor(nTimeout)
-> This calls gpswrapper.gpslocate(), but returns the results rounded to the nearest integer.
