The Graphic Library

The graphics library is a wrapper around paintutils that restores the original cursor position and terminal colours. It also has a text drawing function.

The [tOutput] requires a modded version of paintutils which supports this. It allows pixels to be drawn to other term objects without using term.redirect()

Functions:

graphic.drawFilledBox(nStartX,nStartY,nEndX,nEndY,nColour,[tOutput])
-> Draws a filled box between the start x and y, and the end x and y.
 graphic.drawBox(nStartX,nStartY,nEndX,nEndY,nColour,[tOutput])
-> Draws a hollow box between the start x and y, and the end x and y.
 graphic.drawLine(nStartX,nStartY,nEndX,nEndY,nColour,[tOutput])
-> Draws a line between the start x and y, and the end x and y.
 graphic.drawPixel(nX,nY,nColour,[tOutput])
-> Draws a single pixel at the specified x and y coordinates.
 graphic.drawText(nX,nY,nFG,nBG,sText,[tOutput])
-> Draws text to the screen starting at the specified x and y coordinates, with the specified colour.
